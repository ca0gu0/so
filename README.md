## Linux ssh 登陆工具:

###　一.说明
- 支持秘密和密钥两种格式
- 用户名和密码都是写文件的,明文保存

### 二.配置
- 密码文件配置:

序号:IP:端口:用户:密码:说明
1:192.168.88.128:22:root:toor:虚拟机web服务器

- 密钥文件放在keys文件夹下,密码位置写成密钥文件名,文件名必须以.pem结尾
